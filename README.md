Game Description:
- An infinity runner game where our hero fights an army of nazis and sees everything in pixel art!

Controls:

- Jump: Space Bar
- Shoot: Z

Tactile controls:

- Jump: Touch Left side of the screen.
- Shoot:Touch Right side of the screen.
